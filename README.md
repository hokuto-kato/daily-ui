# 開発
## node
v16.13.2

## dailyIDの切り替え
### デフォルト設定
```
$ yarn default {dailyID}
```
### src → day
```
$ yarn up {dailyID}
```
### day → src
```
$ yarn down {dailyID}
```

## dev
```
$ yarn dev
```

## build
下記のビルドコマンドで`./docs/`内にビルド。  
`yarn up {dailyID}`もしくは`yarn down {dailyID}`で切り替えたIDのディレクトリにビルドされるので注意。
```
$ yarn build
```
